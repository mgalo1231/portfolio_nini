Catsonality
ねこ性格診断サイト

Web Design UI Design

ねこの性格を診断して、タイプでつながる。

作品を見る
期間
2025.11~2026.1(約2ヶ月)
種類
個人制作
担当
企画/UI設計/実装
ツール
Figma/Cursor/Supabase/GitHub

制作プロセス

課題提起

猫の性格を「楽しく・根拠を持って」理解できる診断が少ない。
診断後に日常で活かせる情報まで届けることを課題に設定。

リサーチ

既存サービスを調査し、猫の性格に関する文献を参照。
文献を基準に設問を設計し、診断の納得感を高めた。
飼い主はスマホで情報収集する傾向が高いことも確認。

コンセプト確定

「診断で終わらせず、理解とつながりへ」。
結果を可視化し、共有・コミュニティへ自然に回遊できる体験を目指した。

情報設計

トップ→診断→結果→共有→コミュニティの導線を整理。初見でも迷わないように機能を分けて配置した。

画面設計

余白と大きめCTAで操作負荷を軽減。
やさしい世界観と手描きイラストで親しみやすさを統一した。
ユーザーがスマホで診断する場面を想定し、モバイルファーストで画面を設計。

実装

フロント〜バックエンドを1人で実装（診断ロジック／結果表示／ログイン／投稿）。
体験が途切れない遷移と表示を意識。

振り返り

多頭飼いのプロフィール管理/犬版への展開、タイプ別おすすめ商品連動など、拡張を前提に改善検討中。

学んだこと

プロトタイプ制作初体験 プロトタイプにより早期に完成イメージを共有でき、視覚検証で判断が速くなった。
コンポーネント化で余白・角丸・タイポなどのルールが統一され、実装時の手戻りを削減できた。
1人で前後端を担当 制作範囲が広く難易度は高いが、仕様整理→実装→デバッグまで自走し、
課題に対して解決策を組み立てる力（問題解決力・実行力）が伸びた。
